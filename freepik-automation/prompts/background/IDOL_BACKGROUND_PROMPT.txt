Background Only Prompt Template
Tạo prompt để generate background scene only, không có người/idol

Input dữ liệu:
- background.location
- background.environment_details
- background.depth_and_space
- color_mood.primary_palette
- color_mood.accent_colors
- color_mood.overall_mood
- color_mood.lighting_style
- extra_instructions (optional)

Lưu ý: Template này chỉ tạo background scene, KHÔNG có người, idol, hoặc nhân vật nào trong ảnh/video.

================================================================================
HƯỚNG DẪN ĐIỀN INPUT TỪ JSON
================================================================================

Sau khi có JSON từ Gemini AI phân tích, mapping như sau:

1. Background Information:
   - {background_location} → json.background.location
   - {background_environment_details} → json.background.environment_details
   - {background_depth_and_space} → json.background.depth_and_space

2. Color Mood Information:
   - {primary_palette} → ", ".join(json.color_mood.primary_palette)
     Ví dụ: ["electric blue", "magenta"] → "electric blue, magenta"
   - {accent_colors} → ", ".join(json.color_mood.accent_colors)
     Ví dụ: ["neon green", "warm gold highlights"] → "neon green, warm gold highlights"
   - {overall_mood} → json.color_mood.overall_mood
   - {lighting_style} → json.color_mood.lighting_style

3. Extra Instructions:
   - {extra_instructions_optional} → 
     * Nếu json.extra_instructions có giá trị: json.extra_instructions
     * Nếu json.extra_instructions là null: "" (bỏ qua)

Lưu ý: Chọn template phù hợp với loại background bạn muốn tạo.

   Chọn template theo loại background bạn muốn tạo:
   
   - Template Background Cơ Bản → Cho background tổng quát
   - Template Background cho Vinahouse/Club → Cho club/party background
   - Template Background cho TikTok → Cho social media background
   - Template Background cho Studio → Cho studio/minimalist background
   - Template Background cho Outdoor/Urban → Cho outdoor/street background
   - Template Background cho K-pop Style → Cho stage/performance background

Ví dụ mapping cụ thể:

JSON Input:
{
  "background": {
    "location": "neon-lit rooftop in a modern Asian city",
    "environment_details": "LED strips, city lights in background, urban atmosphere",
    "depth_and_space": "wide open space with cityscape in background, clear depth layers"
  },
  "color_mood": {
    "primary_palette": ["electric blue", "magenta", "deep purple"],
    "accent_colors": ["neon green", "warm gold highlights"],
    "overall_mood": "energetic party",
    "lighting_style": "flashing stage lights"
  },
  "extra_instructions": null
}

Mapping Result:
- {background_location} → "neon-lit rooftop in a modern Asian city"
- {background_environment_details} → "LED strips, city lights in background, urban atmosphere"
- {background_depth_and_space} → "wide open space with cityscape in background, clear depth layers"
- {primary_palette} → "electric blue, magenta, deep purple"
- {accent_colors} → "neon green, warm gold highlights"
- {overall_mood} → "energetic party"
- {lighting_style} → "flashing stage lights"
- {extra_instructions_optional} → "" (vì null)

================================================================================
TÓM TẮT QUY TRÌNH SỬ DỤNG
================================================================================

1. Phân tích background với Gemini AI (nếu cần):
   - Upload ảnh/video background mẫu vào Gemini
   - Hoặc mô tả background mong muốn
   - Nhận JSON output với thông tin background và color_mood

2. Mapping dữ liệu từ JSON vào template:
   - Background: location, environment_details, depth_and_space
   - Color Mood: primary_palette (join array), accent_colors (join array), overall_mood, lighting_style
   - Extra: extra_instructions (nếu có)

3. Chọn template phù hợp:
   - Chọn template theo loại background bạn muốn tạo
   - Có thể dùng template mặc định cho background tổng quát

4. Điền thông tin vào template:
   - Thay thế tất cả các biến {variable} bằng giá trị từ JSON
   - Đảm bảo thêm từ khóa "no people", "empty scene", "no characters"

5. Sử dụng prompt đã điền:
   - Copy prompt hoàn chỉnh
   - Sử dụng với AI generation tool (Kling, Midjourney, DALL-E, v.v.)
   - Để generate background scene only, không có người

Lưu ý quan trọng:
- LUÔN thêm từ khóa: "no people", "empty scene", "no characters", "no idols"
- Template này chỉ tạo background, KHÔNG có người/idol trong ảnh/video
- Đảm bảo JSON có đầy đủ thông tin background và color_mood
- Có thể điều chỉnh template dựa trên overall_mood và lighting_style
