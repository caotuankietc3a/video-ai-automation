Nano Banana Image Generation - Idol + Affiliate Product Prompt Template

Input dữ liệu:
- idol.name
- idol.outfit_description
- idol.pose_style
- idol.body_type
- product.name
- product.category
- product.description
- product.placement
- product.visibility
- background.location
- background.environment_details
- background.depth_and_space
- color_mood.primary_palette
- color_mood.accent_colors
- color_mood.overall_mood
- color_mood.lighting_style
- composition.aspect_ratio (vertical 9:16, square 1:1, etc.)
- composition.framing (full body, half body, close-up, etc.)
- extra_instructions (optional)

================================================================================
HƯỚNG DẪN ĐIỀN INPUT TỪ JSON (GEMINI_IDOL_PRODUCT_ANALYSIS.txt)
================================================================================

Sau khi có JSON từ Gemini AI phân tích ảnh idol + ảnh sản phẩm, mapping như sau:

1. Idol Information:
   - {idol_name_segment} → 
     * Nếu json.idol.name có giá trị: f"{json.idol.name} " (có dấu cách ở cuối)
     * Nếu json.idol.name là null hoặc rỗng: "" (bỏ qua)
   - {idol_outfit_description} → json.idol.outfit_description
   - {pose_style} → json.idol.pose_style
   - {body_type} → json.idol.body_type (nếu cần, thường không dùng trong template)

2. Product Information:
   - {product_name} → json.product.name
   - {product_description} → json.product.description
   - {product_placement} → json.product.placement
   - {product_visibility} → json.product.visibility

3. CHỌN TEMPLATE DỰA TRÊN product.category:
   
   Bước 1: Kiểm tra json.product.category và chọn template theo bảng sau:
   
   ┌──────────────────────┬─────────────────────────────────────────────┐
   │ product.category     │ Template được chọn                          │
   ├──────────────────────┼─────────────────────────────────────────────┤
   │ "clothing"           │ Template cho Quần Áo (Clothing)             │
   │ "accessory"          │ Template cho Phụ Kiện (Accessories)          │
   │ "bag"                │ Template cho Phụ Kiện (Accessories)          │
   │ "footwear"           │ Template cho Phụ Kiện (Accessories)          │
   │ "watch"              │ Template cho Phụ Kiện (Accessories)          │
   │ "jewelry"            │ Template cho Trang Sức (Jewelry)            │
   │ "electronics"        │ Template cho Điện Tử (Electronics)          │
   │ "cosmetics"          │ Template cho Mỹ Phẩm (Cosmetics)            │
   │ Khác (không khớp)    │ Template Tổng Quát (General)                │
   └──────────────────────┴─────────────────────────────────────────────┘
   
   Bước 2: Kiểm tra composition.aspect_ratio để quyết định có dùng format template không:
   
   - Nếu json.composition.aspect_ratio == "9:16" (vertical):
     → Có thể dùng "Template cho Ảnh Dọc TikTok/Instagram Story" 
       THAY VÌ template ở Bước 1 (nếu muốn format chuyên biệt)
   
   - Nếu json.composition.aspect_ratio == "1:1" (square):
     → Có thể dùng "Template cho Ảnh Vuông Instagram Post"
       THAY VÌ template ở Bước 1 (nếu muốn format chuyên biệt)
   
   Bước 3: Nếu muốn sản phẩm nổi bật đặc biệt:
   - Có thể chọn "Template cho Sản Phẩm Nổi Bật (Prominent Product Showcase)"
     cho bất kỳ category nào khi cần highlight sản phẩm là trọng tâm chính
   
   Bước 4: Nếu muốn tạo ảnh idol sexy, gợi cảm, thu hút nam giới:
   - Có thể chọn các "Template cho Idol Sexy" thay vì template category
   - Chọn template sexy phù hợp:
     * "Template cho Idol Sexy + Product" → Cho full body hoặc half body
     * "Template cho Idol Sexy - Close-up Portrait" → Cho close-up, nhấn mạnh mặt
     * "Template cho Idol Sexy - Full Body Showcase" → Cho full body, nhấn mạnh figure
   - Áp dụng khi cần: idol sexy, gợi cảm, thu hút nam giới, emphasize curves
   - Xem thêm phần "CÁCH SỬ DỤNG TEMPLATE SEXY" ở cuối file
   
   Lưu ý:
   - Template format (9:16, 1:1) có thể dùng kết hợp với category template
   - Nếu không chắc, dùng template theo category ở Bước 1
   - Template "Sản Phẩm Nổi Bật" phù hợp khi product.visibility = "prominent" hoặc "featured"
   - Template "Sexy" phù hợp khi cần tạo ảnh gợi cảm, thu hút nam giới

4. Background Information:
   - {background_location} → json.background.location
   - {background_environment_details} → json.background.environment_details
   - {background_depth_and_space} → json.background.depth_and_space

5. Color Mood Information:
   - {primary_palette} → ", ".join(json.color_mood.primary_palette)
     Ví dụ: ["warm beige", "soft white"] → "warm beige, soft white"
   - {accent_colors} → ", ".join(json.color_mood.accent_colors)
     Ví dụ: ["gold highlights"] → "gold highlights"
   - {overall_mood} → json.color_mood.overall_mood
   - {lighting_style} → json.color_mood.lighting_style

6. Composition Information:
   - {composition_framing} → json.composition.framing
   - {composition_aspect_ratio} → json.composition.aspect_ratio
   - Lưu ý: aspect_ratio đã được xử lý ở Bước 2 của phần "CHỌN TEMPLATE"

7. Extra Instructions:
   - {extra_instructions_optional} → 
     * Nếu json.extra_instructions có giá trị: json.extra_instructions
     * Nếu json.extra_instructions là null: "" (bỏ qua)

CÁCH SỬ DỤNG TEMPLATE SEXY:

1. Khi JSON có product.category bất kỳ, bạn có thể thay thế template category
   bằng template sexy tương ứng nếu muốn tạo ảnh gợi cảm.

2. Mapping từ template category sang template sexy:
   - Template Quần Áo → Template cho Idol Sexy + Product (hoặc Full Body)
   - Template Phụ Kiện → Template cho Idol Sexy + Product
   - Template Trang Sức → Template cho Idol Sexy - Close-up (nhấn mạnh trang sức)
   - Template Mỹ Phẩm → Template cho Idol Sexy - Close-up (nhấn mạnh makeup)
   - Template Điện Tử → Template cho Idol Sexy + Product

3. Điều chỉnh các biến trong JSON để phù hợp:
   - pose_style: Thêm "seductive", "alluring", "sensual" vào mô tả
   - outfit_description: Thêm "form-fitting", "revealing", "sexy"
   - overall_mood: Đổi thành "sensual", "alluring", "seductive"
   - lighting_style: Đổi thành "soft romantic lighting" hoặc "dramatic side lighting"
   - composition.framing: Chọn "half body" hoặc "close-up" cho gợi cảm hơn

4. Thêm vào extra_instructions:
   - "emphasizing feminine curves and attractive features"
   - "sensual and alluring expression, seductive gaze"
   - "appealing to male audience, irresistible charm"
